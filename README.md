# Mobile-Application-Testing-and-Reporting


***Mobile Application Testing for Google Keep Notes***

**Overview**
This project contains a manual test plan and test cases for the Google Keep Notes Android application.

**Tools Used**
- Android Studio Emulator
- Google Play Store
- Android Emulator Pixel 9

**Testing Scope**
The focus was on functional testing, UI/UX validation, and basic performance checks.

**Deliverables**
- TestPlan.md
- TestCases.md
- README.md

**Notes**
Testing was performed on an emulator, and results may differ slightly on physical devices.

**All test cases passed during manual testing. No critical defects were found.**
